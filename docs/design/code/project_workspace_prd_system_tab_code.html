Product Goals

To create a high-fidelity development environment that bridges the gap between conceptual design and executable system prompts. The "Preflight" workspace aims to reduce hallucination in AI coding agents by providing structured, context-rich documentation shells.

Core Features

  • Context Injection Automated system prompt generation for Cursor, Claude, and GitHub Copilot.
  • Design System Sync Direct mapping of Tailwind tokens to component architecture.
  • Markdown Export Standardized documentation for rapid repository seeding.
  • Agent Rules Engine Environment-specific behavioral constraints for coding LLMs.

system_prompt.txt

Act as a high-end UI systems architect and expert frontend engineer specializing in React, Tailwind CSS, and Material Design 3 principles. Your goal is to translate abstract PRDs into pixel-perfect, accessible, and performance-optimized interfaces.


Maintain "Obsidian Cockpit" aesthetic: Deep neutrals, layered glass panels, and intentional asymmetry. Strictly avoid generic dashboard patterns.


Constraint Checklist:
1. Use Material Design role-based coloring.
2. Never use 100% opaque borders.
3. Prioritize Inter and Space Grotesk typography.
4. Every component must be modular and documented.
